# Membuat Sistem Login Sederhana

# API Autentikasi Pengguna

Proyek ini adalah API autentikasi pengguna sederhana yang dibangun menggunakan **Node.js** dan **Express.js**. API ini memungkinkan pengguna untuk mendaftar dan login. Data pengguna disimpan dalam file JSON (`users.json`), dan kata sandi disimpan dengan aman menggunakan **bcrypt**.

## Fitur

* Mendaftarkan pengguna baru dengan kata sandi yang di-hash
* Autentikasi pengguna melalui login
* Menyimpan data pengguna dalam file JSON
* Menangani permintaan dan respons dalam format JSON

## Persyaratan

* Node.js (v12 atau lebih tinggi)
* NPM (v6 atau lebih tinggi)

## Catatan

* Kata sandi di-hash menggunakan `bcrypt` dengan salt round `10` untuk meningkatkan keamanan.
* Data pengguna disimpan dalam `users.json` sebagai array objek.
* Kode ini mencakup penanganan kesalahan dasar untuk percobaan login yang tidak valid.

depensi yang di pakai

- express
- nodemon

cara run aplikasi `npm run api`
